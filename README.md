# Resumate

A CLI tool for structuring and managing resume data in a version-controlled format.

## Overview

Resumate helps you organize your career data (experience, projects, skills, education) in structured Markdown files while keeping your workspace clean and accessible.

## Key Features

- **Separated Storage**: Internal tool config (`.resumate/`) is isolated from user data folders
- **Organized Categories**: `experience/`, `projects/`, `skills/`, `education/`
- **Version Control Friendly**: Plain Markdown files
- **AI-Assisted**: Works seamlessly with AI agents
